markdown常用方法

summary
  • 掏粪用

1. 标题

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

2. 区块引用

> 区块引用
>> 嵌套饮用

这里有些绕，是因为用引用呈现出来的

区块引用

嵌套饮用

3. 代码区块

\

前面标注js就是js语言，标注python就是python

4. 强调

*斜体，_斜体_
\*粗体**，__粗体__

单个是斜体，双个是粗体

斜体
粗体

5. 列表

无序列表, · 、+、-

+ 第一项
+ 第二项
+ 第三项

  • 第一项
  • 第二项
  • 第三项

有序列表, 数字加上标点 .

  1. 第一项
  2. 第二项
  3. 第三项
  1. 第一项
  2. 第二项
  3. 第三项

6. 分割线

三个及其以上的 *、 -、_

7. 链接

链接可以由两种形式生成：行内式和参考式。
行内式：

[谷歌](www.google.com.hk)

谷歌

参考式：(本地vscode插件不行)

[谷歌][1]
[苹果][2]
[1]:www.google.com.hk
[2]:www.apple.com

[谷歌][1]
[苹果][2]
[1]:www.google.com.hk
[2]:www.apple.com

8. 图片

![](https://media.52poke.com/wiki/thumb/0/0d/025Pikachu.png/1200px-025Pikachu.png)

9. 符号

`python hello.py`

起到标记作用

python hello.py

10. 表格

| id | name | age |
| – | :–: | –: |
| 3697 | Tomadsafaag | 1024 |
| 2 | Jane | 22 |

注意，第二行的冒号象征着左对齐、居中、右对齐

id name age
3697 Tomadsafaag 1024
2 Jane 22
