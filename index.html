npm-script添加参数

经常使用 npm run [script]指令来避免繁琐命令。
前些天遇到个需求，需要往script里面添加参数，这样就可以复用。

实现

npm run [script] -- [parameter]
注意，双破折号和这个参数之间一定有空格。
eg:

有这么一个script，然后打包Cordova时候需要输入ios还是Android，可以用
npm run copy_to_xcode -- ios
就直接把ios这个参数传递给了指令， 如下
python3 python_cordova.py ios
注意如果script里面有多个指令 && 连接的，参数只会传递给最后一个指令。

markdown常用方法

1. 标题

## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

二级标题

三级标题

四级标题

2. 区块引用

> 区块引用
>> 嵌套饮用

这里有些绕，是因为用引用呈现出来的

区块引用

嵌套饮用

3. 代码区块

\

前面标注js就是js语言，标注python就是python

4. 强调

*斜体，_斜体_
\*粗体**，__粗体__

单个是斜体，双个是粗体

斜体
粗体

5. 列表

无序列表, · 、+、-

+ 第一项
+ 第二项
+ 第三项

  • 第一项
  • 第二项
  • 第三项

有序列表, 数字加上标点 .

  1. 第一项
  2. 第二项
  3. 第三项
  1. 第一项
  2. 第二项
  3. 第三项

6. 分割线

三个及其以上的 *、 -、_

7. 链接

链接可以由两种形式生成：行内式和参考式。
行内式：

[谷歌](www.google.com.hk)

谷歌

参考式：(本地vscode插件不行)

[谷歌][1]
[苹果][2]
[1]:www.google.com.hk
[2]:www.apple.com
